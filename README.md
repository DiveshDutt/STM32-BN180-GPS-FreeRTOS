# STM32-BN180-GPS-FreeRTOS
STM32 firmware utilizing FreeRTOS to read, parse, and process live GPS data streams from a BN-180 GPS module.

## 🚀 Features
- **Multi-tasking Architecture:** Powered by FreeRTOS for efficient task scheduling.
- **GPS Data Parsing:** Extracts NMEA sentences (GPRMC/GPGGA) from the BN-180 module.
- **Serial Communication:** Configured via UART with DMA/Interrupts for non-blocking data transfer.

## 🛠️ Hardware Requirements
- **Microcontroller:** STM32 Series (e.g., STM32F4 / STM32F1)
- **GPS Module:** Beitian BN-180 GPS
- **Interface:** UART/USART connections

## 💻 Software & Toolchain
- **IDE:** STM32CubeIDE
- **RTOS:** FreeRTOS Kernel
- **HAL:** STM32 Cube Hardware Abstraction Layer
